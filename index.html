用css3来实现鼠标悬停改变元素的图标

如右图显示效果

方法一：不用背景图片，用伪元素 + 过渡实现效果

  • 首页
  • 项目
  • 新闻
  • 联系我们

优点:

都是css写出来的效果，减少了页面请求，提高了页面的加载时间

缺点

如果仔细看的话会看到鼠标移入的时候会看到三角形的图标和圆形的图标不是一起显隐的，
如果设置伪元素（三角形）的过渡时间大于元素（圆形）的过渡时间，则鼠标移入时，变化一致，移出时元素（圆形）先隐藏，伪元素（三角形）后隐藏；
如果设置伪元素（三角形）的过渡时间等于元素（圆形）的过渡时间，则鼠标移入时，伪元素（三角形）先显示，鼠标移出时，伪元素（三角形）后隐藏；
如果设置伪元素（三角形）的过渡时间小于元素（圆形）的过渡时间，则鼠标移入时，伪元素（三角形）先显示，鼠标移出时，同时隐藏。

图例显示的是第三种情况

html部分

<ul class="list-one"> <li><span class="text">首页</span></li> <li><span class="text">项目</span></li> <li><span class="text">新闻</span></li> <li><span class="text">联系我们</span></li> </ul>

css部分

.list-one>li { width: 20px; height: 20px; background: rgba(255, 255, 255, .68); border-radius: 50%; position: relative; right: -120px; transition: all .3s; margin: 30px 0; } .list-one>li:after { content: ''; position: absolute; border: 9.3px solid transparent; border-right-color: #3640a7; top: .5px; left: -13px; opacity: 0; z-index: -10; transition: all .1s; } .text { display: none; width: 68px; line-height: 20px; position: absolute; top: 2px; left: -75px; color: #fff; text-align: right; margin-right: 10px; } .list-one>li:hover { background: #3640a7; transform: scale(1.5); } .list-one>li:hover:after { opacity: 1; } .list-one>li:hover .text { display: block; font-size: 10px; }

方法二：用背景图片 + 过渡实现效果

  • 首页

  • 联系我们

优点:

背景图片切换的很流畅

缺点

文字不会跟着放大缩小

图例显示的是第三种情况
